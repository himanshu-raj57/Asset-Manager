# Asset-Manager
# 📑 PDF Asset Manager & Text Extractor

A robust web application built with **Node.js** and **Express** that allows users to upload PDF documents and extract their text content instantly. This tool is designed for quick document analysis and serves as a foundation for a full-scale asset management system.

## 🚀 Features

* **Instant Text Extraction:** Fast and accurate parsing using the `pdf-parse` library.
* **Drag & Drop Interface:** User-friendly UI for seamless file uploads.
* **Metadata Visibility:** View document info like page count and metadata alongside extracted text.
* **Real-time Status:** Backend status monitoring to ensure the server is live and ready.
* **Responsive Design:** Clean, modern interface that works across devices.

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js
- **File Handling:** Multer (Memory Storage)
- **PDF Processing:** pdf-parse
- **Frontend:** HTML5, CSS3, JavaScript (Fetch API)
- **Environment:** Replit Agent / Git
